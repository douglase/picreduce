header_to_FITS_header¶

picreduce.utils.header_to_FITS_header(inputHeader, fmt='hdf5', hdu=None)[source]¶

Takes input header from one format and parses it into a FITS header.

keywords: ‘hdf5’ structured numpy array from a custom PICTURE hdf5 file. ‘idlsave’ header from a custom IDLSAV file opened with scipy.io

Examples

>>> hdu =  header_to_FITS_header(f[im_dset]['sci_header'][0],input='hdf5')
